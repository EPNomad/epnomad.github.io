Introducing Nomad

The API that enables your customers to Discover

Some features you will love!

01.

Train With Just a Few Images

No need to upload hundreds of unique product images, use what you already have. Nomad’s proprietary technology only needs a few images to be trained. As a result, you can train Nomad on 3 or more stock images.

02.

Recognize exact and Visually Similar Products

Nomad not only categories, it specifies. This means Nomad focuses on finding an exact match. Categorization will return a response that does not identify a specific item rather only a group of general items with similar tags such as “red, Nike, sneaker”. These results are often too generic to be useful to a consumer. Specification does not use tags, instead, it utilizes visual similarity. This allows Nomad to return the exact or visually similar products, without depending on logos or text.

03.

Easy to Use API

From a Novice to Expert Developer, Nomad is easy to use. Upload, Train, and Customize Recognition Nodes though an online graphical interface or through code. To implement Nomad API into your product, just add a few lines of code.

04.

Scalability

Nomad solves the industry problem of scalability. Because Nomad looks for visual similarity, it can recognize thousands of products per Recognition Node. These nodes can be linked to create decision trees. This ultimately leads to infinite scalability. No matter how many products you want to upload, Nomad can handle it. All you have to do is plug Nomad’s “Conversion” API into your product database and it will handle the rest.
