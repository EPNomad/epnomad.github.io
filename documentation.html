Nomad Python API Documentation

        

  • Nomad allows anyone to train a neural network to detect items within images.
  • However, for the workshop purposes, we have trained a list of items.

Python example:

  • Example can be found at : https://github.com/dudeyoudontknow/nomad_python_wrapper/
  • To get response :
  • className = Nomad()
  • variableName = className.detect("image.jpeg").text

How to use:

  • You must perform a POST request to our server at the route :

        http://54.208.17.147/zappos/detect/

  • You must convert the image into a Base64-encoded string
  • Send the string through the parameter : { base64: <base64 of image>}
  • The server returns a json response

JSON Response:

  • Example : {"results":[{"name":"teddy bear","confidence":"33"}]}
  • The server returns an array of results named, “results.”
  • The name of the object is returned as, “teddy bear,” and confidence level of 33
